# Cognetix_StudentPerformanceDataset
# Student Performance Dataset

## 📌 Objective
Analyze academic performance to calculate subject averages and identify top students.

## 📊 Dataset
- Source: [Students Performance](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)

## 🛠️ Tools & Libraries
- Python, Pandas, NumPy, Matplotlib, Seaborn

## 🚀 Steps
1. Load dataset and validate data (no nulls/negatives).
2. Compute averages for Math, Reading, Writing.
3. Identify top 5 students by total score.
4. Generate performance visualizations.
5. Extract insights (gender, parental education, ethnicity).

## 📈 Deliverables
- Cleaned dataset
- Averages table
- Top students list
- Visualizations
